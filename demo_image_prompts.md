# Demo Page — Image Generation Prompts

Each scenario has 4 slides. Once images are generated, drop them into the `demo_images/` folder
and swap the `<svg>` placeholder blocks in `demo.html` for `<img src="demo_images/XXX.jpg">` tags.

## SCENARIO 1: Arctic Shield
**Style:** Anduril / comic-book — stark high-contrast, bold ink outlines, halftone dot shadows,
dramatic angles, limited palette: near-black, arctic white, tactical orange (#FF6B35), blood red,
electric green for status readouts. Cinematic, militaristic.

> Comic book panel, Anduril defense aesthetic, high contrast ink art style. Aerial view of Alaskan arctic tundra at night, under a faint aurora borealis. A stark red ALERT banner in the top corner reads "INCURSION DETECTED — SECTOR 7G". In a corner terminal overlay, green monospace text shows an RSE API matching request being fired. Dark navy sky, halftone shadow textures, thick ink outlines. Dramatic, urgent.

## Usage Notes

- For Scenarios 1 and 3, avoid generating anything that looks like real-world people's faces (IP, likeness issues).
- For Scenario 1, lean into the comic aesthetic hard — Midjourney `--style raw` + `--ar 16:9` works well.
- For Scenario 2, try `flat illustration, vector art style` rather than photographic to keep it friendly.
- For Scenario 3, try `DSLR, 85mm, f/1.8, candlelight, editorial food photography` for the food shots.
- Recommended size: 1720×968px (2x of 860×484), saved as progressive JPG at 85% quality.
- Drop finished images into `demo_images/` and update `demo.html` to reference them.
