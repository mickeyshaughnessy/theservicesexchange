The RSE

Request robot labor. Simple demand app — talks only to the public API.

Creates a demand (buyer) account. Provider / robot operators should use the API or website supply tools.

API: rse-api.com · Payment is settled directly with the provider · Not connected to the marketing website.

What do you need done?

Post a priced request. Robot operators match via the exchange.

The exchange does not escrow funds. You pay the provider directly.
